# ✈️ Travel Booking System (Salesforce)

## 📌 Overview

This is a Salesforce-based Travel Booking System built using Apex, Lightning Web Components (LWC), and automation logic. It allows users to create bookings and automatically calculates pricing.

---

## 🚀 Features

* Create travel bookings through UI
* Automatic price calculation using Apex Trigger
* Dynamic user interface using LWC
* Backend logic using Apex
* Scalable and clean architecture

---

## 🛠️ Tech Stack

* Apex
* Lightning Web Components (LWC)
* SOQL
* Salesforce Platform

---

## 📂 Project Structure

* Apex Classes & Triggers
* LWC Components
* Custom Object (Booking__c)
